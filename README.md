Ein simples Plugin für ein OwnerSword 

## Features

- Fügt ein **OwnerSchwert** hinzu

## Kompatibilität

- **Minecraft Version**: 1.21.1
- **Server-Typ**: Spigot

##  Permissions

| Permission | Command        |
|------------|------------------------|
| `Owner`    |`/ownersword` |

## Befehl
**/ownersword**
